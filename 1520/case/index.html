Mike's JS Case Project

Purpose

The purpose of this site was a case project for my JavaScript class at Gwinnett Tech. The Project has two games on the site-Hang Man and Tic Tac Toe which I coded with JavaScript. The site also has a page for an about me section to talk about a little bit more of who I am. I spent some time on this site trying to make a really nice website. I think that it has turned out much better than I originally thought and looks so much better than some of the sites that I was building about 2 years ago! Enjoy!

Game Site

Games

As I mentioned in the above section, this site has two games on it. The first is Hangman and the second is Tic Tac Toe. I made both of these games in the early part of the summer of 2017. I first started with Tic Tac Toe and after some initial struggle with working with JavaScript, I made the game. Next came Hangman that I literally thought about the entire design in the shower! Finally, the games are not fully responsive so they don't look nice on mobile devices...sorry!

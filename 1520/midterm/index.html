MidTerm Project

Requirements

Create a Web page that contains a form. On the form create five elements in which you can enter integers. Create a function that instantiates the five elements of the array from the form elements and then sums the five integers using the array and a for loop and prints the answer on the page as text. Call the function when the form is submitted. Be sure to include enough text on the form so that what is to be done is clear. ie "Please enter integers".
